Proyecto Evaluación 3 - Aplicación Flask

Este proyecto contiene una aplicación web desarrollada en Flask que incluye:

- Menú principal con dos ejercicios
- Formulario de notas con validación y cálculo de promedio
- Formulario de nombres con comparación de longitud
- Estilo CSS personalizado
- Validaciones y comentarios en el código
